📌 VBA (Visual Basic for Applications)란?

Excel, Word 등 MS Office 프로그램 내에서 반복 작업을 자동화하고, 맞춤형 기능을 만들 수 있게 해주는 프로그래밍 언어예요. 버튼 하나로 복잡한 HTML 태그 입력을 대신 처리할 수 있습니다.

VBA 자동화를 통해 기존 수작업 대비 업무 효율성을 6배 증가시키는 성과를 달성했습니다. (시험지 1개당 30분 → 5분)

2. 개발배경

[ 업무 상황 ]
NELT & 교재 성취 TEST 등 디지털로 구현되는 문항 개발 시 HTML 태그 작업이 필요했습니다. 한 시험지 기준 (평균 30문항) 평균 30분 이상의 시간이 소요되었으며, 이는 반복적이고 단순하지만 연구원이 직접해야 하는 업무였습니다.

[ 비효율 ]
또한, 어드민 시스템에 등록한 후에야 태그 오류 확인이 가능했습니다. 오류가 발견되면 개별 문항을 일일이 찾아 수정하고, 수정 작업 후 재확인 과정을 반복하는 비효율이 발생했습니다.

VBA 매크로 로직 설계

💡 VBA 매크로 핵심 로직
- 필요한 엑셀 파일/워크시트/열 선택
- 선택한 열의 데이터(밑줄, 이탤릭, 볼드 등)를 기반으로
- 오른쪽 열에 HTML 태그가 자동 입력된 데이터 출력

[미리보기 기능]
- 필요한 엑셀 파일/워크시트/열 선택
- 웹(HTML)으로 문항 미리 보기 창 자동 생성
- 오류 문항을 엑셀 파일에서 찾아 수정 후 즉시 재확인
