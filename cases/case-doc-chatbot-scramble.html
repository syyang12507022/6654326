📌 AI 문장 청킹(Chunking)이란?

AI(LLM)가 문장의 구조를 분석하여 '의미 단위(청크)'로 자동 분할하는 기술이에요. 사람이 직접 주어구, 동사구, 목적어구 등을 판단할 필요 없이, 챗봇이 알아서 문장을 나눈 뒤 순서를 섞어(스크램블) 학습 자료를 만듭니다.

1. 성과요약

AI 챗봇 자동화를 통해, 10~20개의 영어 문장을 의미 단위(청크)로 나누고 스크램블하는 데 소요되던 **수작업 시간을 획기적으로 단축**시켰습니다. 생성된 자료는 직독직해 학습용, 워크시트 제작 등 다양하게 활용 가능해졌습니다.

➡️ 영어 학습 자료(청크/스크램블) 제작 시간 대폭 단축!

2. 개발배경

[ 업무 상황 ]
영어 학습 자료 제작 시, 10개~20개 가량의 영문장을 의미 단위 청크로 하나하나 나누고 스크램블(순서 섞기)하는 작업이 필요했습니다.

[ 비효율 ]
PDF 지문을 워드로 복사한 뒤, 담당자가 직접 영문장을 확인하고 주어구/동사구/목적어구/전치사구로 판단하여 나누는 수작업에 시간이 많이 소요되었습니다.

AI 자동화 핵심 로직

💡 자동화 이전/이후 프로세스
[자동화 이전 (수작업)] 1. PDF 파일 지문을 워드로 복사 2. 문장 단위로 분리 3. 담당자가 직접 문장을 보며 주어구/동사구/목적어구 등 의미 단위(청크)로 판단하여 나누는 작업 4. 나뉜 청크를 수동으로 스크램블(순서 섞기) [자동화 이후 (AI 챗봇)] 1. 챗봇에게 영어 문장(들)을 입력 2. (챗봇) 문장을 의미 단위(청크)로 자동 분할 3. (챗봇) 분할된 청크를 자동 스크램블하여 결과 제공

AI 챗봇 실행 화면

챗봇에게 문장을 입력하고 '의미 단위로 청킹한 뒤 스크램블해줘'라고 요청하면, 즉시 결과물을 생성합니다.
